save-pdf

API for pdf (master branch)

by Unknown

Usage: 
(ns your-namespace
  (:require incanter.pdf))

Overview


This library currently has only a single function, save-pdf, which saves
charts as a PDF file. To build this namespace make sure the you have the iText
library (http://itextpdf.com/) as a declared dependency in your pom.xml or 
project.clj file:
[com.lowagie/itext "1.4"]

save-pdf

  function

  
Usage: (save-pdf chart filename & options)


  
 Save a chart object as a pdf document.

Arguments:
  chart
  filename

Options:
  :width (default 500)
  :height (defualt 400)

Examples:
  
  (use '(incanter core charts))
  (save-pdf (function-plot sin -4 4) "./pdf-chart.pdf")
